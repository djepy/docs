# Plugins installed at the start of every Claude Code on the web session.
# One per line. Format: <plugin-name>[@<marketplace-name>]
# Blank lines and lines starting with # are ignored.
#
# The marketplace name is the name declared in its marketplace.json, NOT the
# GitHub repo. For jeremylongshore/claude-code-plugins the marketplace name is
# `claude-code-plugins-plus`.
#
# To add a plugin: append `<name>@claude-code-plugins-plus` on its own line,
# commit, and push. The next session will install it automatically.

# Auto-loads AGENTS.md files into the agent's context when relevant.
agent-context-manager@claude-code-plugins-plus

# Adds the /commit slash command that drafts a conventional-commit message
# from the current git diff.
ai-commit-gen@claude-code-plugins-plus

# Ruflo — full plugin suite from the ruvnet/ruflo marketplace (32 plugins).
# Marketplace name in marketplace.json is `ruflo` (the @suffix below).
#
# WARNING: ruflo-core registers PreToolUse + PostToolUse + Stop hooks that
# fire on every Bash / Write / Edit / MultiEdit call and on session end.
# The shim calls `ruflo hooks <subcmd>` and falls back to
# `npx --prefer-offline --yes ruflo@alpha hooks ...` if the CLI isn't on
# PATH — i.e. it can auto-install the alpha CLI silently on first fire.
# It also intentionally swallows stderr and forces exit 0, so hook failures
# are invisible. If/when this becomes a problem:
#   claude plugin disable ruflo-core@ruflo
# ─────────────────────────────────────────────────────────────────────────────
